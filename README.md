# Langchain-Projects🚀

This repository is part of my journey exploring LangChain and building practical, hands-on AI applications. The goal is to understand how LLM-powered systems can be designed using tools like prompts, agents, tools, and retrieval systems to create intelligent and scalable solutions.

---

## 📌 Projects Included

### 🎬 Movie Summarizer
A movie data extraction tool that:  
- Extracts structured information from a given paragraph  
- Outputs data strictly in JSON format  
- Includes fields like title, cast, genre, plot, rating, etc.  

**Use Case:**  
Converts unstructured movie descriptions into structured data for analysis or storage.

---

### 🔍 CodeAnalyser
An AI-powered Python code analysis tool that:  
- Analyzes code for bugs, performance, and readability  
- Suggests improvements  
- Can execute code safely and return outputs  
- Uses tool-based reasoning via agents  

**Use Case:**  
Helps developers debug, optimize, and understand Python code.

---

### 🤖 Copilot
An advanced AI research assistant that:  
- Uses agents with tools like web search and Python execution  
- Performs multi-step reasoning using ReAct agents  
- Generates structured research reports  
- Supports memory and iterative reasoning  
- Includes a Streamlit UI for interaction  

**Use Case:**  
Acts as an intelligent assistant for research, analysis, and report generation.


### 📚 RAG (Retrieval-Augmented Generation)
An advanced system that combines retrieval with generation.

- Uses external knowledge sources  
- Improves response accuracy  
- Context-aware answers  

---

**Use Case:** Building intelligent QA systems with real-world data.

---

### 🖥️ UI Bot
An interactive chatbot with a user interface.

- Chat-based interaction  
- User-friendly UI  
---
**Use Case:**  Provides an intuitive way to interact with LLM applications.

- LangChain
